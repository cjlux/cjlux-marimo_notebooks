# 🇬🇧 *Bordeaux Python Meetup 17 juin 2025*<br>Marimo : Python Notebooks to Replace Jupyter Notebooks?
 This repository contains the slides and notebooks presented on June 17th at the "Bordeaux Python Meetup" on the topic "Marimo: Python Notebooks to Replace Jupyter Notebooks?".<br>
To use the notebooks:
- install the Python project manager "uv" (see https://docs.astral.sh/uv/getting-started/installation/)
- download the zip file from the GitHub repository (green "Code v" button)
- unzip the archive and in a console (Linux/MacOS terminal or Windows PowerShell) navigate to the `marimo_notebooks-master` folder with the command:<br>
`cd <path to marimo_notebooks-master>`<br>
where `<path to marimo_notebooks-master>` is the path to the `marimo_notebooks-master` folder, which you can retrieve with an application such as a file browser.
- Type the `uv sync` command to install the Python modules into the repository's virtual Python environment.
- Launch marimo by typing:
`uv run marimo edit`
